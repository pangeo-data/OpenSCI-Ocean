# Code of Conduct / 行为准则

## 核心原则

1. **尊重**：尊重每一位参与者的时间、专业背景和贡献方式
2. **诚实**：对 AI 的使用诚实披露，对数据和结果诚实报告
3. **建设性**：技术批评对事不对人，提出问题的同时给出建设性建议
4. **包容**：欢迎不同学科背景、不同经验水平的参与者

## 学术诚信

- 不编造或篡改数据和结果
- 不抄袭他人成果（包括未经许可使用他人未发表的 idea）
- AI 生成的内容必须经过人工审查，不能未审核直接使用
- 文献引用必须真实可查

## 沟通规范

- GitHub Issue 和 PR 中使用中文或英文均可
- 讨论技术问题时提供证据和数据支持
- 对他人的代码和分析结果给出具体、可操作的反馈
- 不在公开渠道讨论未发表的核心数据细节

## 违规处理

轻微违规（如沟通不当）：由项目负责人私下提醒。
严重违规（如学术不端）：取消参与资格，相关贡献记录保留但署名权取消。
