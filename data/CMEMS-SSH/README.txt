CMEMS SSH 数据下载步骤
========================

1. 注册 CMEMS 账号:
   https://data.marine.copernicus.eu/register

2. 安装 Python 3.8+（当前环境是 Python 3.7，copernicusmarine 需要 3.8+）
   或直接用网页下载：
   https://data.marine.copernicus.eu/product/SEALEVEL_GLO_PHY_L4_MY_008_047/download

3. 需要的参数:
   产品: SEALEVEL_GLO_PHY_L4_MY_008_047
   变量: adt, ugosa, vgosa
   区域: 南纬 40°–75°
   时段: 1993-01–2025-06
   格式: NetCDF
